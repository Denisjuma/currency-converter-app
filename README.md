# Currency Converter App

![App Version](img.shields.io)
![Flutter Version](img.shields.io)
![License](img.shields.io)

Hii ni programu rahisi ya kubadilisha sarafu iliyotengenezwa kwa kutumia Flutter. Programu inabadilisha viwango vya ubadilishaji kutoka **Dola ya Marekani (USD)** kwenda **Shilingi ya Tanzania (TZS)** kwa kutumia API ya viwango vya ubadilishaji.

## Vipengele Muhimu (Features)

*   **Ubadilishaji wa Haraka:** Badilisha USD kwenda TZS papo hapo.
*   **Data Halisi:** Inatumia [Currency Freaks API](currencyfreaks.com) kupata viwango vya sasa vya ubadilishaji.
*   **Kiolesura Safi:** Muundo wa kisasa na rahisi kutumia (Material Design).

Fuata hatua hizi ili kuendesha mradi huu kwenye mashine yako:

### 1. Clone Repository
